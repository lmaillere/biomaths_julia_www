Matériel supplémentaire


Les pages listées ici contiennent du matétiel supplémentaire pour compléments plus avancés :

  • améliorer la rapidité d’exécution des simulations en utilisant au maximum la définition de struct et de function

  • tracer une jolie figure pour Lotka Volterra avec Makie.jl

  • positivité: simuler un modèle en forçant le shéma numérique à respecter sa positivité

  • semi-discret: simuler un modèle semi-discret/impulsionnel via l’interface callback de DifferentialEquations.jl

  • réaliser des graphiques animés avec Makie.jl, un exemple sur Lotka Volterra

  • simuler une équation de réaction-diffusion en une dimension : le modèle de Fisher-KPP

Pour aller plus loin

Nous n’avons qu’effleuré l’utilisation de Julia, par exemple tout le code présenté est scripté et relativement peu de code est inclus dans des fonctions. Nous n’avons pas non plus abordé les types struct (à part dans cette page) et le multiple dispatch qui sont pourtant au coeur du fonctionnement, de l’efficacité, et de la versatilité de ce langage.

Pour aller plus loin (et aussi revoir quelques bases !), vous pouvez par exemple vous référer à ces tutoriels:

  • le cours Introduction to computational thinking du MIT (assez long)

  • le workshop Julia Zero-to-Hero de Georges Datseris (plus direct).

Annexes

Pour tester d’autres interfaces d’intégration:

  • utiliser l’interface integrator de DifferentialEquations.jl ou la redéfinition de problèmes d’intégration via remake

## Pour aller plus loin

Nous n'avons qu'effleuré l'utilisation de `Julia`, par exemple tout le code présenté est scripté et relativement peu de code est inclus dans des fonctions. Nous n'avons pas non plus abordé les types `struct` (à part dans cette [page](annexe_swift_rma.qmd)) et le `multiple dispatch` qui sont pourtant au coeur du fonctionnement, de l'efficacité, et de la versatilité  de ce langage.



Pour aller plus loin (et aussi revoir quelques bases !), vous pouvez par exemple vous référer à ces tutoriels:

- le cours [Introduction to computational thinking](https://computationalthinking.mit.edu/Fall23/) du MIT (assez long)

- le workshop [Julia Zero-to-Hero](https://github.com/Datseris/Zero2Hero-JuliaWorkshop) de Georges Datseris (plus direct).
